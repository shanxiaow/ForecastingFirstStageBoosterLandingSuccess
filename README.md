## Forecasting First-Stage Booster Landing Success

SpaceX has transformed commercial spaceflight by making rockets partially reusable. The defining innovation of these rockets is that the first-stage booster can return and land vertically after launch, allowing it to be reused instead of discarded. The successful reusability dramatically lowers launch costs and enables an unprecedented launch frequency. Predicting first‑stage landing outcomes is essential for mission planning, cost modeling, and understanding the operational reliability of reusable rockets.

This project builds an end‑to‑end data science pipeline to predict whether the first stage will land successfully, using data collected from SpaceX’s public API and web‑scraped launch records.



### Project Contents:  
 *  Collect data using SpaceX REST API and web scraping  
 *  Wrangle data to prepare for analysis
 *  Explore data with data visualization and SQL
 *  Build interactive Folium map to examine launch site geography
 *  Build interactive Plotly Dash dashboard to explore the importance of launch site and payload effects
 *  Build and evaluate four machine learning classification models to predict landing outcome  



### Results Summary
- Identified key predictors of landing success
- Compared performance across multiple ML models
- Visualized launch site importance and payload effects
- Created interactive tools for exploring the dataset
